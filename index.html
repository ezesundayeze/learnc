A Practical 30-Day C Course

Welcome! This repository contains a series of lessons designed to teach you the C programming language from the ground up, with a focus on practical application, mental models, and modern best practices.

Course Modules

  • Module 0: Your First Steps in C (Days 1-2)
  • Module 1: Variables and Basic I/O (Days 3-5)
  • Module 2: Making Decisions and Repeating Actions (Days 6-9)
  • Module 3: Building Reusable Blocks of Code (Days 10-13)
  • Module 4: Pointers and Memory - The Heart of C (Days 14-17)
  • Module 5: Working with Text - C Strings (Days 18-20)
  • Module 6: Creating Custom Data Types with Structs (Days 21-23)
  • Module 7: Dynamic Memory Allocation (Days 24-26)
  • Module 8: Building Larger Projects (Days 24-26)
  • Module 9: Interacting with the System (Days 27-29)
  • Module 10: Capstone Project - Inventory Management System (Days 30-34)
